## Step 1 вЂ” Get a Merriam-Webster API Key (Free)

1. Go to https://dictionaryapi.com/register/index
2. Register for a **free** Collegiate Dictionary key
3. Open `api/config.php` and replace `YOUR_MERRIAM_WEBSTER_API_KEY`

---

## Step 2 вЂ” Set Up Firebase

1. Go to https://console.firebase.google.com
2. Create a new project (e.g., `searchy-dict`)
3. **Authentication** в†’ Enable **Email/Password** provider
4. **Firestore Database** в†’ Create database (start in **test mode**, then apply rules below)
5. **Firestore** в†’ Rules tab в†’ paste contents of `firestore.rules` в†’ Publish

---

## Step 3 вЂ” Connect Flutter to Firebase

Install FlutterFire CLI (once):
```bash
dart pub global activate flutterfire_cli
```

Run inside the project:
```bash
flutterfire configure --project=YOUR_FIREBASE_PROJECT_ID
```

This overwrites `lib/firebase_options.dart` with your real config.

---

## Step 4 вЂ” Deploy the PHP Backend

### Option A вЂ” Local (Android Emulator)
1. Install XAMPP / WAMP / Laragon
2. Copy the `api/` folder to your `htdocs` folder (e.g., `htdocs/searchy_api/`)
3. In `lib/utils/constants.dart` set:
   ```dart
   const String kApiBaseUrl = 'http://10.0.2.2/searchy_api';
   ```
   (`10.0.2.2` is localhost from inside the Android emulator)

### Option B вЂ” Live Server
1. Upload the `api/` folder to your web host (e.g., `https://yourdomain.com/searchy_api/`)
2. Update `kApiBaseUrl` in `lib/utils/constants.dart`

---

## Step 5 вЂ” Run the App

```bash
flutter pub get
flutter run
```

## Features
- **Register / Login** with email & password (Firebase Auth)
- **Password reset** via email
- **Search** any English word (Merriam-Webster Collegiate)
- **Definitions, pronunciation, part of speech, etymology**
- **Auto-save** every search to your account (Firestore)
- **History screen** вЂ” tap any word to re-search it
- **Swipe to delete** individual history entries
- **Clear all** history with confirmation dialog
- **Guest mode** вЂ” search without an account (no history saved)
